## 🚀 서비스 기능

### 1. 회원가입

- 등록된 아이디인지 체크
- 비밀번호입력 재확인 체크

### 2. 로그인

- 아이디, 비밀번호 체크
- 아이디, 비밀번호 일치하면 메인페이지 렌더링
- 로그인 정보 local storage 저장

### 3. 메인 (스케쥴러)

- Calendar 구현(Vanilla JS)
- 등록된 여행일정  마킹
  - 여행 제목 표시
  - 여행타입(나홀로, 가족, 커플, 친구)에 따른 마킹컬러 구분

### 4. 일정 추가 페이지

- 여행 제목, 기간, 타입, 일정 리스트 등록

### 5. 마이페이지

- 등록된 일정 목록 렌더링
- 목록 클릭시 세부일정 모달창
- 일정 삭제 기능
